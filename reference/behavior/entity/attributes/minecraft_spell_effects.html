minecraft:spell_effects

コンポーネントが追加されたときにエンティティに追加または削除するステータスエフェクトを設定します

パラメーター

add_effects

"add_effects": [] //リスト(配列,中身はオブジェクト)

デフォルトの値:

コンポーネントが追加された時にエンティティに追加するエフェクトのリスト

※エフェクトの順番はそのエフェクト達が付与されている状態の時、エフェクト一覧で表示される「下から」順に書かないと正常に動作しない可能性があります。

"add_effects" リストのパラメーター {

effect

"effect": "文字列"

エンティティに付与するエフェクトの名前

名前は/effect コマンド時に使用するエフェクトの名前です(多分)

duration

"duration": 数値

エンティティに付与するエフェクトの秒数

1で1秒

amplifier

"amplifier": 数値

エンティティに付与するエフェクトのレベル

4以上にするとエフェクトのレベルが表示されなくなります

ambient

"ambient": ?

現在不明わかる人がいれば教えていただけるとありがたいです…

visible

"visible": 真偽値

エフェクトが付与された時に出てくるパーティクルの表示設定

trueで表示, falseで非表示

} "add_effects" リストのパラメーター終了

remove_effects

"remove_effects": "文字列"
または
"remove_effects": [
  "文字列",
  "文字列",
  ....
]

コンポーネントが追加されたときにエンティティから削除するエフェクトの名前(のリスト)

名前は/effect コマンド時に使用するエフェクトの名前です(多分)

エフェクトの順番は関係有りません

配列にすると指定されたエフェクトを全て削除します

remove_effectsはadd_effectsの「前」に実行されます

サンプルコード

"minecraft:spell_effects": {
  "add_effects": [
    {
      "effect": "speed",
      "duration": 120
    }
  ]
}

コンポーネントが追加されたときにエンティティに「移動速度上昇(レベル4)を2分間」付与する

"minecraft:spell_effects": {
  "add_effects": [
    {
      "effect": "night_vision",
      "duration": 240,
      "visible": false
    },
    {
      "effect": "haste",
      "duration": 240,
      "amplifier": 3,
      "visible": false
    },
    {
      "effect": "speed",
      "duration": 120,
      "amplifier": 3,
      "visible": false
    }
  ]
}

コンポーネントが追加されたときにエンティティに「暗視(レベル1)と採掘速度上昇(レベル4)を4分間、移動速度上昇(レベル4)を2分間 "パーティクル非表示" 」で付与する

"minecraft:spell_effects": {
  "remove_effects": "speed"
}

コンポーネントが追加されたときにエンティティから「移動速上昇」を削除する

"minecraft:spell_effects": {
  "remove_effects": [
    "speed",
    "poison",
    "night_vision"
  ]
}

コンポーネントが追加されたときにエンティティから「移動速上昇,毒,暗視」を削除する

"minecraft:spell_effects": {
  "add_effects": [
    {
      "effect": "haste",
      "duration": 120
    }
  ],
  "remove_effects": [
    "speed",
    "night_vision"
  ]
}

コンポーネントが追加されたときにエンティティから「移動速度上昇,暗視」を削除し、「採掘速度上昇(レベル1)を2分間」付与する

バージョンが違っていたりすると動作が違う・使用できない可能性があります。ご注意ください
