minecraft:attack

エンティティの近接攻撃力と攻撃時の追加効果を設定します

パラメーター

"damage": [数値, 数値]
もしくは
"damage": 数値

デフォルトの値:

近接攻撃時の与えるダメージの値

配列にすると 「数字の大きい方以上 数字の小さい方未満」 の値をランダムで与える

値をマイナスにすると金ハートが最大8個まで増えていきます-100にしても8個までです

通常のハートは増えません

小数点以下は切り捨てられます

"effect_duration": 数値

デフォルトの値: 0.0

攻撃したエンティティに与えるステータスフェクトの秒数

1で1秒

小数点以下は切り捨てられます

"effect_name": "文字列"

デフォルトの値:

近接攻撃されたエンティティに適応されるステータスエフェクトの名前

名前は/effect コマンド時に使用するエフェクトの名前です(多分)

サンプルコード

"minecraft:attack": {
  "damage": 5
}

近接攻撃時5ダメージを与える

"minecraft:attack": {
  "damage": [5, 20]
}

近接攻撃時5以上20未満ダメージを与える

"minecraft:attack": {
  "damage": [5, 20],
  "effect_duration": 5,
  "effect_name": "blindness"
}

近接攻撃時5以上20未満ダメージを与え、「目潰し」を5秒付与する

バージョンが違っていたりすると動作が違う・使用できない可能性があります。ご注意ください
