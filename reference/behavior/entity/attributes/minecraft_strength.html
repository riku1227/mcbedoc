minecraft:strength

アイテムを運ぶエンティティの持てるアイテムのスロット数を設定します

※「minecraft:inventoryのcontainer_typeがhorse」「additional_slots_per_strengthが3以上」でないと正常に動作しません

パラメーター

"max": 整数値

エンティティの持てる最大スロット数を設定します

1で「minecraft:inventoryのadditional_slots_per_strength」で指定した数値スロットです

"value": 整数値

エンティティの持てるスロット数を設定します

maxで指定した値よりも値が大きかったとしてもmaxの値になります(例:maxが3,valueが5だとvalueが3として扱われる)

1で「minecraft:inventoryのadditional_slots_per_strength」で指定した数値スロットです

※「minecraft:inventoryのinventory_size」÷「additional_slots_per_strength」を超えると、スロットは表示されますが、アイテムを入れても保存されずそのアイテムが消えます

サンプルコード

"minecraft:inventory": {
  "inventory_size": 256,
  "container_type": "horse",
  "additional_slots_per_strength": 3
},

"minecraft:strength": {
  "value": 3,
  "max": 5
}

additional_slots_per_strengthが3の場合

エンティティの最大スロット数は15

エンティティのスロット数は9

"minecraft:inventory": {
  "inventory_size": 256,
  "container_type": "horse",
  "additional_slots_per_strength": 5
},

"minecraft:strength": {
  "value": 7,
  "max": 7
}

additional_slots_per_strengthが5の場合

エンティティの最大スロット数は35

エンティティのスロット数は35

バージョンが違っていたりすると動作が違う・使用できない可能性があります。ご注意ください
