minecraft:ambient_sound_interval

エンティティが再生する鳴き声などの音が再生される間隔を設定します

パラメーター

"range": 数値

デフォルトの値: 16.0

音が再生されるまでの最大遅延時間を設定します

0〜指定した値のランダムで実行されます(人力で計測したので正確な検証はできていません)

小数点いかがどうなるのかは検証できていません

1で1秒です

"value": 数値

デフォルトの値: 8.0

エンティティが再び音を再生するまでの最小時間を設定します

実際にエンティティが再び再生するまでの時間は 「valueで指定した値 + [0〜rangeで指定した値のランダム]」秒です

小数点いかがどうなるのかは検証できていません

1で1秒です

サンプルコード

"minecraft:ambient_sound_interval": {
  "range": 6.0
}

「8 + [0〜6のランダム]」秒に一回エンティティが音を再生する

"minecraft:ambient_sound_interval": {
  "value": 1.0
}

「1 + [0〜16のランダム]」秒に一回エンティティが音を再生する

"minecraft:ambient_sound_interval": {
  "range": 12.0,
  "value": 5.0
}

「5 + [0〜12のランダム]」秒に一回エンティティが音を再生する

"minecraft:ambient_sound_interval": {
  "range": 0.0,
  "value": 1.0
}

1秒に一回エンティティが音を再生する

バージョンが違っていたりすると動作が違う・使用できない可能性があります。ご注意ください
